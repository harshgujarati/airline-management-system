# Airline Management System

A relational database system for managing airline operations built with MySQL.

## Features
- 10 normalized tables (Travelers, Flights, Reservations, Payments, Crew, etc.)
- Views for passenger reservations and journey schedules
- Stored procedures for automated booking and payment processing
- Triggers for auto-cancellation on payment failure
- User-defined functions for luggage weight and revenue calculations

## Database Schema
- Traveller
- Operation_Manager
- Fleet_Aircraft
- Air_Journey
- Reservation
- Boarding_Pass
- Crew_Member
- Crew_Assignment
- Payment
- Luggage

## Technologies Used
- MySQL
- SQL (DDL, DML, Views, Stored Procedures, Triggers, Functions)
